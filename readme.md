# GoodsGarage

[Live Demo](https://goods-garage.web.app/)

## Description

GoodsGarage is a full-stack web application that connects users worldwide to export and import products based on their business or personal needs.  
Users can explore product details, add their own exports, and import items shared by others — all within an intuitive and responsive interface.

## Features

- User authentication and login system  
- View detailed information about products  
- Add new products or exports  
- Import products added by other users  
- Product quantity management and availability tracking  
- Toast notifications and confirmation alerts for smooth UX  

## Technologies Used / Stack

- **React** — for building the user interface  
- **React Router** — for client-side routing and protected routes  
- **Tailwind CSS** — for modern, responsive styling  
- **daisyUI** — for ready-made, aesthetic UI components  
- **React Toastify** — for toast notifications  
- **SweetAlert2** — for beautiful success/error modals  
- **Express.js** — for backend API  
- **MongoDB** — for database and product storage
